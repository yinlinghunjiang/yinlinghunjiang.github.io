    FurBot-灵梦

    简介

    一个综合性兽圈bot [开源仓库]

    申请机器人请填写表单,填写完后加机器人好友并邀请机器人进入相应群聊

    我才不会告诉你bot的名字来自东方project

    注意事项

    请确保群可被搜索，使用本机器人默认你同意了 《用户协议》 《隐私政策》

    我们不允许私自将机器人拉进群聊，以免造成误会

    指令

    写在前面

    指令框中 方括号[] 为可选参数， 尖括号<> 为必选参数

    指令中无需把括号写进去

    指令 描述 示例
    .help /help help（以上都可以触发） 获取bot使用说明 /
    来只毛 用于随机获取一只毛毛的图片。 /
    来只 <毛毛名字> 用于随机获取一只特定名字的毛毛。毛毛名字可以写缺少字符，Bot会自动使用模糊查找，匹配含有这个字的毛毛。 来只毛 果糖
    找毛图 查毛图 用于按FurID查询毛毛的图片。 查毛图 1
    每日鉴毛 用于随机获取一只毛毛的每日鉴毛 /
    <毛毛名字> 的每日鉴毛 用于获取一只特定名字的毛毛的每日鉴毛 果糖 的每日鉴毛
    <期数> 期每日鉴毛 用于获取特定期数的每日鉴毛 1期 每日鉴毛
    查云黑 <号码> 用于按QQ查询兽圈云黑 查云黑 1581647952
    签到 每日签到 /
